# 🤖 ML Regression Studio

ML Regression Studio is a Streamlit-based machine learning web application that allows users to upload datasets, clean data, train regression models, evaluate model performance, and download prediction results without writing any code.

---

## ✨ Features

- 📁 Upload CSV Dataset
- 🧹 Handle Missing Values
- 🗑 Remove Duplicate Records
- 🎯 Feature Selection
- 🤖 Train Multiple Regression Models
  - Linear Regression
  - Multiple Linear Regression
  - Polynomial Regression
- 📊 Model Evaluation
  - R² Score
  - MAE
  - MSE
- 📈 Actual vs Predicted Graph
- 📥 Download Prediction Results

---

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/ML-Regression-Studio.git
```

Go to project folder

```bash
cd ML-Regression-Studio
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

---

## 📷 Application Workflow

1. Upload Dataset
2. Clean Dataset
3. Select Regression Model
4. Train Model
5. View Performance Metrics
6. Visualize Predictions
7. Download Prediction Results
